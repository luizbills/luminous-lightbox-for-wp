== Description ==

A very lightweight image lightbox plugin based on [Luminous](https://github.com/imgix/luminous) by imgix.

### How to use

- Just link your images to **Media File** (see in *Screenshots* Section).

### Useful Links:

- [Plugin Github Repository](https://github.com/luizbills/luminous-lightbox-for-wp)
- [Luminous Lightbox Demo](https://imgix.github.io/luminous/)

== Installation ==

Installing "Luminous Lightbox for WordPress" can be done either by searching for "Luminous Lightbox for WordPress" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
